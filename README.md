# Sales Dashboard (2025–26)
## Project Overview
This project presents an interactive Sales Dashboard developed using Microsoft Power BI.
The dashboard helps analyze sales performance using multiple KPIs and visualizations.
---
## Dashboard Features
- Total Sales KPI
- Total Cost KPI
- Total Quantity KPI
- Total Transactions
- Category-wise Quantity Analysis
- State-wise Sales Map
- Brand-wise Sales Distribution
- Supervisor Filter
---
## Tools Used
- Microsoft Power BI
- Microsoft Excel
